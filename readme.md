## Conference API

> A simple Lumen API for talks at PHP Conference Brazil 2019


### About
- An endpoint for Talks  
- An endpoint for Speakers

### Stack: 

#### Backend
- PHP 7.2^
- Lumen 6
- Composer

#### Database
- MySQL 5.8^

---

### Installation

#### Prerequisite
- config file `.env`
- create local database  

#### Backend
- $ composer install
